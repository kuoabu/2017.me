RebbiJet 睿傑控房系統

RebbiJet is a online prototype management system for hoteliers, features Booking Calendar, Channel Manager and Smart Lock, etc. We are devoted to creating a revolution in hotel and travel chain.

This is the biggest system I have ever been responsible for alone.

My duty is to design and develop the presentation-layer and front-end architecture of the website. It is a challeng for me to controll so many events and data in the same time between users and server. So I developed the system by using ReactJS and Redux, and thinking in React's one-way data flow. On the part of css, I used Sass to develop responsive design website and to be more adaptive to the changes.
